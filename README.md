## Игра «Города» для Яндекс Алисы

Обучающий проект для Яндекс.Лицея — игра с Алисой, где игроки по очереди называют города на последнюю букву.

При создании проекта использовался датасет городов от [arbaev/russia-cities](https://github.com/arbaev/russia-cities).
