## Игра «Города» для Яндекс Алисы

Обучающий проект для Яндекс.Лицея — игра с Алисой, где игроки по очереди называют города на последнюю букву.

В проекте используется база городов России от [arbaev/russia-cities](https://github.com/arbaev/russia-cities).

### Пояснения по фразам для Алисы
#### Формат фразы
Алиса отвечает короткими фразами, в которых места для подстановки значений обозначены "{}"
#### Пример фразы
"Я назвову город {}", {} - имя города
#### Какие фразы нужны
- приветсвие, объяснение правил, вывод доступных диалоговых опций
- назвать город + сказать пользователю, что теперь его очередь
- принять город пользователя + назвать город
- город пользователя не найден
- пользователь уже называл этот город
- прислать изображения города на карте по просьбе пользователя
- объявить о победе пользователя если Алиса не смогла ответить
- объявить о победе Алисы если пользователь сдался

_Для каждой фразы необоходимо создать 2-3 вариации._
